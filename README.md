## Install

Step 1.

    npm install inventive
    or
    bower install inventive
    or
    git clone git@github.com:speedovation/Inventive.git 

Step 2.

    npm install
    bower install
    gulp 

Step 3.
    
    That's it. Start development.

### CSS Components

* Grid
* Tables
* Forms
* Buttons
* Tooltips
* Messages
* Badges
* Thumbnail
* Navbar
* List Group
* Nav
* Subnav
* Panel
* Breadcrumb
* Pagination
* Progress
* Search
* Mega menu
