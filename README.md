## Install

Step 1.

    npm install inventive
    or
    bower install inventive
    or
    git clone git@github.com:speedovation/Inventive.git 

Step 2.

    npm install
    bower install
    gulp 

Step 3.
    
    That's it. Start development.

* Buttons

* Thumbnail

* List Group

* Progress
