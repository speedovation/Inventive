Responsive Navbar

Simple yet effective navbar

Important helpers

We can add class .fixed to make it fix in current position.

Also we can add class .top to make it fix at top. This class won't work without .fixed class.

Always add .fixed and .top class to nav element only.

Javascript

We need to initialize nav. No support for data attributes in this version.

 $(".flexnav").flexNav();
