A design focused front-end framework using Stylus, Jade, CoffeeScript and Gulp

Welcome in our responsive world! An introduction of Speedovation, getting started , downloads and use, basic templates and examples, and more.

Introduction

What is Inventive?

Inventive is an open source project created by Speedovation

Inventive is a complete Front-end framework using Stylus, CoffeeScript and Jade.

Clean and Slick

Clean and slick themes

Modern and clean theme for components which make it easier to customize and lightweight.

Stylus based

Stylus is robust, expressive, and feature-rich CSS superset

Stylus is a revolutionary new language, providing an efficient, dynamic, and expressive way to generate CSS. Supporting both an indented syntax and regular CSS style.

Features

Inventive is a powerful framework.

It provides lots of css and js components. Themeable and fully responsive. Super powerful grid system. Slick look. Execellent typography.

Modular

Inventive is built based on the modularity provided by CSS3 and enhanced by Stylus.

Inventive composed of modules. Css components and Js components. Easily add in add in projects. Keep overall project lightweight.

Contribute

Inventive invites everyone.

Make a fork of the project on Github and start contributing to this young project. You can contribute in coding, documentation, community support.

Browser Support

Inventive Framework is tested and supported in major modern browser

Chrome, Firefox, Opera, Safari and Internet Explorer.

Powerful grid

Mobile first and fully responsive
