Responsive Utilities

Responsive utilities can show and hide content based on devices. Print utility also included.

Responsive classes

Use a single or combination of the available classes for toggling content across viewport breakpoints.

Mobiles (<768px) Tablets (≥768px) Desktops (≥992px) Large (≥1200px)
.visible-mobile-* Visible Hidden Hidden Hidden
.visible-tablet-* Hidden Visible Hidden Hidden
.visible-desktop-* Hidden Hidden Visible Hidden
.visible-large-* Hidden Hidden Hidden Visible
.hidden-mobile Hidden Visible Visible Visible
.hidden-tablet Visible Hidden Visible Visible
.hidden-desktop Visible Visible Hidden Visible
.hidden-large Visible Visible Visible Hidden

.visible-*-* classes for each breakpoint come in three variations, one for each CSS display property value listed below.

Group of classes CSSdisplay
.visible-*-block display: block;
.visible-*-inline display: inline;
.visible-*-inline-block display: inline-block;

So, for mobiles ( mobile ) screens

For example, the available .visible-*-* classes are:

  • .visible-mobile-block
  • .visible-mobile-inline
  • .visible-mobile-inline-block
